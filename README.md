<h1>json-server usando Postman</h1>

> Status: Concluído ✨

### API Fake

* Instalei as dependências NPM
* Criei uma API - Fake
* Depois com json-server --watch efetuei testes na Api Fake 
* Utilizei o Postman e nele exercitei os métodos:
* GET
* POST
* PUT
* PATCH
* DELITE

### Tecnologias utilizadas:

* NPM 
* Visual Stúdio Code
* Json-Server
* JavaScript
* Postman
